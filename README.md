# Entrega Tarea 1 - Sistemas Recomendadores

## 1. Estructura de archivos

Nuestra entrega consiste de 4 archivos:

### 1.1 Informe (.pdf)
Explica detalladamente nuestro análisis exploratorio de datos, la implementación de los algoritmos de recomendación solicitados en el enunciado y, finalmente, una discusión profunda sobre los resultados obtenidos.

### 1.2 `t1_analisis_exploratorio.ipynb`
Notebook que contiene todo nuestro código para el análisis exploratorio realizado en ambos datasets: Yahoo! Music y Wikimedia.

### 1.3 `t1_algoritmos_yahoo.ipynb`
Notebook con la implementación de los algoritmos necesarios para la parte de Yahoo! Music. Incluye todos los resultados para uKNN, iKNN, FunkSVD, Random y Most Popular.

### 1.4 `t1_algoritmos_wikimedia.ipynb`
Notebook con el código para la parte de Wikimedia. Contiene los resultados para los algoritmos ALS, BPR y Content Based.

## 2. Detalles para replicar

Los notebooks cuentan con todas las celdas ejecutadas, no obstante, el ayudante puede volver a ejecutar de manera secuencial las celdas para reproducir los resultados presentados en el informe. El unico requisito para esto es importar el archivo de datos de la tarea: `OneDrive_2026-03-25.zip`.
